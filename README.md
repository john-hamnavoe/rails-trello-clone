# README

Very basic trello clone.

Uses:

* Ruby version 2.7.0

* Rails Version 6.1.0

* Stimulus 2.0

* Sortable js

* Bootstrap

* acts_as_list gem

* Devise - there but not hooked up.

* Hotwire - turbo

* Action Cable

* Database Postgresql 

  * rails db:migrate

* No tests 

* Run:

  * Deploy Database
  * rails s
